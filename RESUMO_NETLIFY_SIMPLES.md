## 🎯 3 PASSOS SIMPLES

### **PASSO 1: Preparar Pasta (5 min)**

```
1. Crie pasta: C:\dashboard-netlify\

2. Copie ficheiros para lá:
   ├─ index.html (novo)
   ├─ setup.html (renomeado)
   ├─ admin.html (renomeado)
   ├─ dashboard.html (renomeado)
   └─ dados_teste.json (copia)

3. Pronto! Pasta com 5 ficheiros.
```

### **PASSO 2: GitHub (5 min)**

```
1. Abra: https://github.com
2. "New repository"
3. Nome: dashboard-financeiro
4. "Create repository"

5. Clique "Add file" → "Upload files"
6. Arraste a pasta dashboard-netlify para cima
7. Commit changes

8. URL: https://github.com/SEU_USERNAME/dashboard-financeiro
```

### **PASSO 3: Netlify (10 min)**

```
1. Abra: https://netlify.com
2. Sign up com GitHub (autorize)

3. "New site from Git"
4. Escolha "dashboard-financeiro"
5. Clique "Deploy site"

6. Espera 1 minuto...
7. Recebe URL: https://dashboard-financeiro-xyz.netlify.app
8. ✅ PRONTO! Site funcional!
```
