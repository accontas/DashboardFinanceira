🎯 Como Usar

Primeira Vez (Configuração): Clique em "Setup" para guardar o ID da pasta Google Drive
Mensalmente (Dados): Clique em "Admin" para carregar o CSV do mês
Sempre (Visualizar): Clique em "Dashboard" para ver os dados atualizados
✅ Tudo funciona automaticamente. O cliente apenas precisa de abrir o Dashboard!
