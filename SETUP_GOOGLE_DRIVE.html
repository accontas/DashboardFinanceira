📊 Setup Dashboard com Google Drive

Sincronização Automática - Cliente Apenas Visualiza

🔧 Passo 1: Preparar Google Drive

Este é um processo que faz uma única vez. Depois é automático!

1. Criar Pasta em Google Drive
Aceda a drive.google.com e:
  • Clique em "Novo" → "Pasta"
  • Dê nome: "Dashboard-Financeiro-Dados"
  • Guarde a pasta vazia (vamos usar para guardar JSONs)
2. Criar Ficheiro JSON de Exemplo
Dentro da pasta que criou, crie um ficheiro de teste:
dashboard_dados.json
Este ficheiro será atualizado automaticamente com os seus dados.
3. Partilhar Pasta com Cliente
Clique com botão direito na pasta → "Partilhar"
  • Escolha "Acesso de Editor" ou "Visualizador"
  • Partilhe com o cliente (email dele)
  • Guarde o link de partilha

⚙️ Passo 2: Admin.html (Você - Carrega Dados)

Este é o ficheiro que você usa para carregar dados mensalmente.

O que faz:
✅ Carrega CSV do software contábil
✅ Valida e processa dados
✅ Guarda em localStorage (seu PC)
✅ Sincroniza automaticamente em Google Drive
✅ Cliente vê dados atualizados
Como Usar (Rotina Mensal):
1. Abra dashboard_financeiro_admin_googledrive.html no seu PC
2. Carregue CSV do mês
3. Clique "Guardar Dados"
4. Clique "Sincronizar com Google Drive"
5. Pronto! Cliente já vê dados atualizados
✅ Nota: O Admin.html ainda não está preparado com Google Drive. É um próximo passo que vou criar.

📊 Passo 3: Dashboard.html (Cliente - Visualiza)

Este é o ficheiro que o cliente abre no navegador.

O que faz:
✅ Abre em navegador (Netlify ou link público)
✅ Automaticamente busca dados de Google Drive
✅ Mostra todos os dados (gráficos, tabelas, KPIs)
✅ Sem botões de upload
✅ Sem forma de editar dados
✅ Sempre mostra dados mais recentes
Como Cliente Usa:
1. Você envia link: https://seu-site.netlify.app/dashboard.html
2. Cliente abre no navegador
3. Vê todos os dados automaticamente
4. Seleciona período que quer (mês/trimestre/ano)
5. Ponto, fim! Nada mais para fazer
✅ Nota: O Dashboard.html actual já tem suporte para isto. Vou preparar a versão com Google Drive.

🔗 Passo 4: Configurar URLs Google Drive

Para que Admin e Dashboard se comuniquem via Google Drive:

Encontra na URL do Drive: https://drive.google.com/drive/folders/[AQUI]
